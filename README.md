# Advent of Code

## Usage

```bash
# Generate code for day `1` of `2024` and fetch input (note example will be empty)
gleam run new --fetch --year=2024 1 --example
# Run a day `1` of `2024` with the `--example` input
gleam run run --year=2024 1 --example
# Run a day `1` of `2024` with the real input
gleam run run --year=2024 1
```
